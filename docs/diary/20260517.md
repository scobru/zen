## 5. ROOT CAUSE TÌM ĐƯỢC: udpSay bypass WebSocket

**(Checkpoint 015 — commit `5c62d9b`)**

Đọc kỹ `src/mesh.js` `mesh.hear["relay"]`:

```js
// Trước fix — relay direct forward:
if (p.udpSay) { try { p.udpSay(fwd); return; } catch(e) {} }
// ... WebSocket code bên dưới KHÔNG BAO GIỜ chạy nếu udpSay có mặt

// Trước fix — relay flood:
if (fpeer.udpSay) { try { fpeer.udpSay(fwd); continue; } catch(e) {} }
```

**UDP không đáng tin cậy**. Khi UDP packet bị mất → message lost **silently**. WebSocket (TCP) bị skip hoàn toàn nhờ `return`/`continue`.

zen↔peer1 có UDP fast path (`[UDP] Fast path for 0kiu2ZCd → peer1.akao.io:8421`) → zen dùng UDP → packet loss → peer1 không nhận.

peer0→peer1 lúc trước pass vì peer0 không có UDP fast path cho peer1 → fallthrough tới WebSocket.

**Fix:**

```js
// Sau fix — chỉ dùng WebSocket cho relay forwarding:
mesh.say(fwd, peer);    // direct forward
mesh.say(fwd, fpeer);   // flood
```

**Kết quả ngay lập tức: 6/6 hop tests PASS** ✅

**Commit:** `5c62d9b` — `fix: relay forwarding — use WebSocket only, not UDP`

## 6. Fix Node.js v18 ESM crypto issue

**Commits:** `b18cb04` rồi reverted bởi `38981b1`

**Vấn đề**: peer1 service dùng Node.js v24 (qua nvm), nhưng E2E tests chạy với system node v18. Trong Node.js v18 ESM file module, `globalThis.crypto` là `undefined` → `api.random` crash.

```bash
# Chứng minh:
node -e "console.log(typeof globalThis.crypto)"  # "object" (CJS, OK)
echo "console.log(typeof globalThis.crypto)" > t.mjs && node t.mjs  # "undefined" (ESM, BROKEN)
```

**Fix thử nghiệm**: Inject polyfill vào đầu `zen.js` bundle:

```js
if (typeof globalThis !== 'undefined' && typeof globalThis.crypto === 'undefined') {
  try { globalThis.crypto = (await import('node:crypto')).webcrypto; } catch (_) {}
}
```

Commit `b18cb04` — hoạt động cho peer1 tests. **Nhưng**: `npm run test:mcp` break với `ERR_REQUIRE_ASYNC_MODULE` vì mocha v6 dùng `require()`, không load được async ESM module.

**Revert** với `38981b1`: xóa polyfill. Justification:
- Cả 3 relay servers dùng Node.js v24 (nvm), không cần polyfill
- Servers import `index.js` chứ không phải `zen.js`  
- Browsers có native crypto
- Tests phải chạy với Node.js v24 (`source ~/.nvm/nvm.sh && nvm use 24`)
